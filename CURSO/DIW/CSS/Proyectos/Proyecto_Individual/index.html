Manual para Configurar Servidores Web para Ejecutar Java EE


Este manual tiene como objetivo explicar cómo configurar tres servidores web
para ejecutar aplicaciones Java EE, incluyendo Servlets, JSP, WebSockets y EJBs,
en sistemas operativos Windows y Linux.

Se cubrirán tres servidores distintos que pueden ser utilizados en entornos de desarrollo web:
Apache Tomcat, WildFly y GlassFish.
